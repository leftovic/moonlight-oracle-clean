\# Moonlight Oracle



Moonlight Oracle is a cozy nightly ritual web app where users draw mystical cards, receive soft reflective readings, and return daily for a calming bedtime experience.



\## MVP Features

\- Daily draw

\- Yes/No oracle

\- 3-card spread

\- Local streak tracking

\- Ambient sound

\- Responsive UI



\## Stack

\- Next.js

\- Tailwind CSS

\- Local JSON content

\- localStorage

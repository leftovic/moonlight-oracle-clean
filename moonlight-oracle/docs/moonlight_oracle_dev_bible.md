# Project Vision

Moonlight Oracle is a **cozy nightly ritual app** where users draw
mystical cards, receive comforting insights, and relax with ambient
soundscapes.

Core emotional goals:

* calm
* curiosity
* ritual
* mystical atmosphere
* short repeatable engagement

Primary loop:

Open app → draw card → reveal message → reflect → return tomorrow
